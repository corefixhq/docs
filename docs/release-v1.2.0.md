# CoreFix v1.2.0

**Release Date:** June 23rd, 2026.

## Added

- **Scanner Logs to R2** — scanner logs are now pushed to R2 for storage.
- **Live Scanner Logs in Frontend** — scan logs now stream live to the frontend as scans run.

## Changed

- **ZAP policy tuning** — updated max scan durations, max rule durations, and spider coverage.

## Bug Fixes

- Fixed bugs in API key generation, affecting both CI/CD pipeline usage and local runs.
- Fixed bugs in adding or inviting members to workspaces.
- Fixed bugs in creating new workspaces.
