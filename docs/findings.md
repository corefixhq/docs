## Unified Security Findings

## Overview

The Unified Security Findings dashboard in CoreFix serves as the central command center for vulnerability management. It consolidates security findings across your infrastructure into a single, intelligent interface-eliminating fragmented visibility and reducing manual triage effort.

By bringing vulnerabilities, misconfigurations, and risks together in one place, the platform enables faster decision-making, improved prioritization, and streamlined remediation workflows. This unified approach ensures your team spends less time navigating tools and more time resolving critical issues.

---

## Key Capabilities

- Centralized visibility across all discovered vulnerabilities  
- Intelligent prioritization to focus on high-impact risks  
- Advanced search and filtering for rapid investigation  
- Direct access to technical evidence and remediation guidance  
- Integration with public vulnerability standards such as CWE and CVE  

---

## Navigating the Unified Dashboard

The dashboard is designed for clarity and operational efficiency. Each finding is presented with actionable context, enabling security teams to move from detection to resolution without friction.

### Search and Discovery

Use the search bar to quickly locate specific vulnerabilities across your environment. This is particularly useful when investigating known issues or validating fixes.

### Filtering Insights

Leverage built-in filters to refine results based on severity, asset type, or other parameters. This allows teams to isolate high-priority threats and reduce noise during analysis.

### Timestamp Awareness

Each finding includes a discovery timestamp, helping teams understand when the issue was identified and prioritize remediation timelines accordingly.

---

## Understanding Findings

Each vulnerability entry provides layered insights designed to support both high-level assessment and deep technical analysis.

### Quick View Information

- Hover over a finding to view associated URLs  
- Identify port numbers directly from the hover interaction  
- Quickly assess context without navigating away from the dashboard  

### Detailed View

Click on a finding to access a comprehensive breakdown, including:

- Severity level  
- Affected components  
- Remediation guidance  
- Risk context  

### Extended Analysis

- Select **Details** to access an expanded summary of the vulnerability  
- Review deeper insights that support informed decision-making  

---

## Security Classification and Standards

CoreFix aligns findings with globally recognized security standards to enhance clarity and traceability.

### CWE Classification

Each finding includes a CWE (Common Weakness Enumeration) identifier. Clicking on the CWE ID provides insight into the vulnerability category and its broader security implications.

### CVE Mapping

Where applicable, findings are mapped to CVE (Common Vulnerabilities and Exposures) identifiers. These links connect directly to public vulnerability databases, enabling further research and validation.

## Technical Evidence and Outputs

To support remediation and verification:

- Click on **Output** to access raw scan data and technical evidence  
- Review actionable insights derived from the scan engine  
- Use this information to validate vulnerabilities and guide fixes
