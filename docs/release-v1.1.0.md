# CoreFix v1.1.0

**Release Date:** June 12th, 2026

## Added

- **API Key Creation** for authenticating CLI, CI/CD, and API-based scans.
- **Workspace Creation** to organize projects and scans.
- **Inviting Users to Workspaces** for team collaboration.
- **Triggers** to kick off scans based on defined conditions.
- **Cron Jobs** for scheduling recurring scans.
- **CI/CD Pipeline Integrations** for running scans as part of build pipelines.
- **Free Credits** — new accounts receive up to $10 in free credits.
- **Credit Calculation** — runtime credits and AI credits are now calculated and tracked for every scan.
