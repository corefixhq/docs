## Scanners Available

### Infrastructure-as-Code Scanner

Detects security vulnerabilities and misconfigurations in IaC files including:

* Terraform
* Kubernetes manifests
* Dockerfiles
* Helm charts

The scanner identifies:

* Security misconfigurations
* Compliance violations
* Insecure infrastructure definitions
