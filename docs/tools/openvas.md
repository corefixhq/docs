## Scanners Available

### Complete Vulnerability Assessments

Performs deep vulnerability scanning of systems to identify known security issues.

The scanner:

* Checks devices with installed security agents
* Compares detected services and packages against vulnerability databases
* Identifies known vulnerabilities using CVE databases

This helps organizations ensure **patch compliance and vulnerability remediation** across infrastructure.
