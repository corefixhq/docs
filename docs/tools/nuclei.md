## Scanners Available

### Rapid Web Threat, Misconfig & CVE Scanner

A comprehensive vulnerability scanner that detects security issues across:

* Web applications
* Networks
* Infrastructure

It uses a large library of community-maintained templates to identify:

* Known vulnerabilities (CVEs)
* Security misconfigurations
* Exposed services
