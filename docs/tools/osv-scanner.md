## Scanners Available

### OSV Vulnerability Scanner

Identifies security issues in open-source libraries and third-party packages.

The scanner cross-references dependencies against the **Google OSV database** to detect:

* Vulnerable packages
* Known CVEs
* Security advisories

## Scanners Available

### Software Composition Analysis (SCA)

Identifies vulnerabilities in **open-source dependencies and third-party packages** used in the application.

The scanner analyzes project dependency files to detect:

* Vulnerable libraries
* Outdated packages
* Known security issues in dependencies
