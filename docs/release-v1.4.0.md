# CoreFix v1.4.0

**Release Date:** July 4th, 2026.
## Added

- **HAR Recording** — all recorded web traffic is now viewable in a new HAR dashboard in the frontend.
- **SBOM integration** in the code scanner, with results now shown in a dedicated SBOM dashboard in the frontend.
- **Package Health scoring**, covering:
  - Supply chain health
  - Open source vulnerabilities
  - Secrets
  - Infrastructure health
  - Code health / code smells
  
  These signals are combined into a single, collective health score.
- **Labs Dashboard** in the frontend, featuring one-click deployment of popular vulnerable-by-design apps (WebGoat, Juice Shop, DVWA, etc.) as Vercel sandboxes for quickly testing web application security capabilities.
